# apps/mobile

App mobile para pacientes (React Native + Expo + NativeWind). **Será migrado do repositório `noun-app`.**

## Status

Placeholder — aguardando migração.

## Para migrar

1. Copiar o conteúdo de `noun-app` para esta pasta
2. Atualizar imports para usar `@noun/types`
3. Configurar o `tsconfig.json` para usar `@noun/config/tsconfig/react-native`
4. Verificar compatibilidade de versões do Expo SDK

## Requisitos de desenvolvimento

- Expo CLI: `npm install -g expo-cli`
- Android Studio (para Android) ou Xcode (para iOS)
- Expo Go no dispositivo físico para testes rápidos
